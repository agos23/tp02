1.1¿Qué es una URL?

una url es una cadena de texto que te lleva a un recurso de internet.

1.2- ¿Qué es una url absoluta?

una url absoluta ordena y dispone la url completa incluido el protocolo (http: // o https: //).

1.3- ¿Qué es una url relativa?

Una URL relativa es aquella que describe la ruta a un archivo relativo al actual documento,se pueden usar cuando se vincula a otro documento se pueden usar cuando se vincula a otro documentoy solo requiere delnombre de la ruta.

1.4-¿Cuál es la diferencia entre una url relativa y una url absoluta?

la diferencia entre una url relativa y una url absoluta es que la relatiba describe la ruta de un archibo ,mientras que la absoluta ordena la url y el protocolo

1.10- ¿Cuál es el propósito de la etiqueta anchor?

la etiqueta anchor crea un enlace a paginas de internet ,archivos dentro de la pagina o cualquier otra URL.

1.11- ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

se utiliza el atributo href que es un atributo de la etiqueta anchor que contiene componentes como La URL.

1.12- ¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

si este atributo esta vacio no funciona.

1.13- ¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

Algunos valores para target son: self: carga la URL en el contexto actual _blank: carga la URL en un nuevo contexto de navegación.es una pestaña, los usuarios pueden configurar los navegadores para utilizar una ventana nueva en lugar de la pestaña.

1.14- Entonces ¿qué es un hipervínculo?

un hipervinculo es un elemento de un documento electrónico que se refiere a otro recurso, por ejemplo, a otro punto dentro del mismo o de otro documento.
